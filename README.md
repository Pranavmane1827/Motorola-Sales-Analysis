
📊 Motorola Sales Analysis Dashboard – Power BI

🔹 Short Description / Purpose
Developed an interactive Power BI dashboard to analyze Motorola mobile sales performance across cities, products, and time. The dashboard provides insights into sales trends, customer behavior, and transaction patterns for data-driven business decisions.

🛠️ Tech Stack
Power BI
DAX (Data Analysis Expressions)
Power Query
Excel / CSV

📂 Data Source
Sales dataset containing:
City
Mobile Model
Sales Amount
Quantity
Transactions
Payment Method
Customer Ratings
Date (Monthly data)

✨ Features / Highlights
Interactive month-wise filtering
KPI cards (Total Sales, Quantity, Transactions, Average Sales)
City-wise sales visualization using map
Product-wise performance comparison
Customer rating analysis
Payment method distribution insights

🎯 Goal
To provide a comprehensive view of sales performance, enabling businesses to identify trends, optimize strategies, and improve customer satisfaction.

📊 Key Visuals
KPI Cards: Total Sales (64M), Quantity (2K), Transactions (326), Avg Sales (40K)
Map Chart: Sales distribution across cities in India
Line Chart: Daily quantity trends
Bar Chart: Sales by mobile models (iPhone, Redmi, Vivo)
Pie Chart: Transactions by payment method (UPI, Card, Cash)
Funnel Chart: Customer ratings distribution
Table: Brand-wise sales and quantity

💼 Business Impact
Identifies top-performing cities and products
Tracks customer preferences and payment trends
Helps optimize inventory and sales strategy# Motorola-Sales-Analysis
